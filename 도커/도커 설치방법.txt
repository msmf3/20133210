도커 설치(윈도우)

https://hub.docker.com/editions/community/docker-ce-desktop-windows
도커 회원가입 후 설치

도커는 리눅스 컨테이너이므로 맥이나 윈도우즈엔 가상머신에 설치되는 개념임.
윈도우즈(클라이언트) - 리눅스(서버) 가상 서버에 설치된 도커가 동작하는 이유.

가상화
docker는 윈도우에선 Hyper-V라는 가상화 기술을 사용한다.
가상화 enabled 시켜야 함.
BIOS 설정 들어가서 가상화 on

실행할 때 메모리 부족한 현상
윈도우10에 램 4GB이면 Docker 세팅에서 Memory 를 2048MB -> 1024MB 로 바꿔줘야 한다. 
( 4GB에서 기존에 사용하는 메모리가 있기 때문에 Docker가 차지하는 메모리 최소화. )
